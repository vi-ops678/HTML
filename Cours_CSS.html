Introduction

CSS (Cascading Style Sheets) est un langage utilisé pour décrire l'apparence des pages web en HTML. CSS permet de styliser les éléments HTML en modifiant leur couleur, taille, position, etc.

Structure de base d'un fichier CSS

Un fichier CSS contient des règles de style qui sont appliquées aux éléments HTML. Chaque règle se compose d'un sélecteur et d'une déclaration de style entre accolades.

h1 {
    color: blue;
    font-size: 24px;
}

Les sélecteurs CSS

  • Sélecteur de type : Sélectionne tous les éléments d'un certain type, comme les balises HTML. Par exemple, p sélectionne tous les paragraphes.
  • Classe : Sélectionne tous les éléments ayant une classe spécifique. Par exemple, .ma-classe sélectionne tous les éléments ayant la classe ma-classe.
  • ID : Sélectionne un élément ayant un ID spécifique. Par exemple, #mon-id sélectionne l'élément avec l'ID mon-id.

Les propriétés CSS

Les propriétés CSS sont les règles que vous appliquez aux éléments HTML. Par exemple, pour changer la couleur du texte, vous utilisez la propriété color.

  • color: Définit la couleur du texte.
  • font-size: Définit la taille de la police.
  • background-color: Définit la couleur de fond d'un élément.
  • padding: Ajoute de l'espace à l'intérieur d'un élément.
  • margin: Ajoute de l'espace à l'extérieur d'un élément.
  • border: Définit une bordure autour d'un élément.

Exemple de style CSS

Voici un exemple de fichier CSS pour styliser une page HTML :

/* Appliquer une couleur de fond à la page */
body {
    background-color: #f0f0f0;
}

/* Styliser les titres */
h1 {
    color: blue;
    font-size: 24px;
}

/* Ajouter une bordure autour des paragraphes */
p {
    border: 1px solid black;
    padding: 10px;
}
