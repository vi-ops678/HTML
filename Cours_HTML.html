Introduction

Le HTML (HyperText Markup Language) est le langage de base utilisé pour créer des pages web. Il sert à structurer le contenu d'une page en utilisant des éléments appelés "balises".

Structure de base d'un document HTML

Un document HTML commence toujours par une déclaration de type HTML, suivie d'une structure de base avec des balises importantes telles que ``, ``, et ``.

<!DOCTYPE html>
<html lang="fr">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Titre de la page</title>
    </head>
    <body>
        <h1>Bienvenue</h1>
        <p>Voici un paragraphe</p>
    </body>
</html>

Les principales balises HTML

  • <h1> à <h6> : Balises pour les titres, où <h1> est le plus grand et <h6> le plus petit.
  • <p> : Balise pour un paragraphe de texte.
  • <a> : Balise pour créer un lien hypertexte.
  • <img> : Balise pour insérer une image.
  • <ul> : Balise pour créer une liste non ordonnée.
  • <ol> : Balise pour créer une liste ordonnée.
  • <li> : Élément d'une liste.

Exemple de lien et d'image

Voici comment insérer un lien vers une autre page HTML :

<a href="https://www.example.com">Cliquez ici pour visiter notre site</a>

Et voici comment afficher une image :

<img src="image.jpg" alt="Description de l'image">
