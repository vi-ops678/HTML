Cours JavaScript - Bloc1-Prog

Introduction

JavaScript est un langage de programmation principalement utilisé pour rendre les pages web interactives. Il permet de manipuler le contenu HTML et CSS pour créer des effets dynamiques.

Structure de base d'un programme JavaScript

Un programme JavaScript peut être intégré directement dans une page HTML via la balise <script>. Le code JavaScript peut également être écrit dans un fichier externe, lié à la page HTML.

<script>
    console.log("Hello, World!");
</script>

Les variables en JavaScript

Les variables permettent de stocker des valeurs que l'on peut manipuler dans un programme. JavaScript utilise les mots-clés let, const, et var pour déclarer des variables.

  • let : Permet de déclarer une variable dont la valeur peut être modifiée.
  • const : Permet de déclarer une variable dont la valeur ne peut pas être modifiée une fois définie.
  • var : Ancienne méthode de déclaration, moins utilisée de nos jours.

Les fonctions en JavaScript

Une fonction est un bloc de code qui peut être exécuté à tout moment. Elle peut accepter des paramètres et retourner des valeurs.

function direBonjour(prenom) {
    console.log("Bonjour, " + prenom);
}

direBonjour("Alice");

Les conditions en JavaScript

Les conditions permettent d'exécuter certaines parties du code en fonction de tests. Par exemple, avec if et else.

let age = 18;
if (age >= 18) {
    console.log("Vous êtes majeur.");
} else {
    console.log("Vous êtes mineur.");
}

Les boucles en JavaScript

Les boucles permettent de répéter une action plusieurs fois. Par exemple, la boucle for permet d'itérer sur une séquence de valeurs.

for (let i = 0; i < 5; i++) {
    console.log(i);
}
